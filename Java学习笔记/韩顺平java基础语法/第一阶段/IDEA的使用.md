# IDEA 的使用

## 快捷键

- ## （1）运行：`ctrl + shift +F10`
  - ## 自己设置：`alt + R`
- ## （2）生成：`alt + inesert`
- ## （3）格式化代码：`ctrl + alt + L`
- ## （4）删除当前行（==删除当前行内容后会把后面的内容往上补充==）
  - ## 默认是：`ctrl + Y`
  - ## 自己设置：`ctrl + D`
- ## （5）复制当前行
  - ## 自己设置：`ctrl + alt + 向下箭头`
- ## （6）补全代码（提示消失之后需要使用代码补全）：`alt + /`
- ## （7）注释：`ctrl + /`
- ## （8）查看继承关系（把光标放在类名上）：`ctrl + H`
- ## （9）快速跳转到某个方法或者调用的方法（光标放在对应的方法上）：`ctrl + 鼠标左键单击`，或者使用`ctrl + B`
- ## （10）自动分配变量名：`.var + 回车`
- ## （11）自动导入该类需要的包（先要在设置中配置 auto import 的相关选项）：`alt + 回车`
