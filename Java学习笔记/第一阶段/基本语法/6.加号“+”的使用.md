# 加号 `+` 的使用

# 1. ==字符串的拼接==

## 示例代码

```java
public class hello{
    public static void main(String[] args){
        System.out.println("jackson"+"在"+"学Java");
    }
}
```

## 输出：`jackson在学Java`

# 2. 输出加法的结果

```java
public class hello{
    public static void main(String[] args){
        System.out.println(10+10);
    }
}
```

## 输出：`20`

# 3. 混合使用（区别以下二者）

## 案例一

```java
public class hello{
    public static void main(String[] args){
        System.out.println(100+3+"hello");
    }
}
```

## 输出：`103hello`

## 案例二

```java
public class hello{
    public static void main(String[] args){
        System.out.println("hello"+100+3);
    }
}
```

## 输出：`hello1003`

## 注意：计算机的运行是从左到右，从上到下逐步运行的
